PROJECT: Mini-GPT from Scratch (for "Attention Is All You Need" Midterm Presentation)
======================================================================================

GOAL:
  Build a miniature GPT (decoder-only transformer) in PyTorch, train it on text data,
  and use the results in a 12-15 minute midterm presentation covering problem motivation,
  method, experiments/analysis, and insights from the paper.

  [x] GPU verified: NVIDIA A40, 46GB VRAM, CUDA working
  [x] Training data downloaded: shakespeare.txt (TinyShakespeare, ~1MB)
  [x] Write config.py -- hyperparameters (batch_size, block_size, n_embd, n_head, etc.)
  [x] Write model.py MultiHeadAttention class -- fused K/Q/V projections (N_EMBD -> N_EMBD),
      reshape+transpose to split into heads, scaled dot-product attention, causal mask,
      dropout, output projection (Linear N_EMBD -> N_EMBD)
      (supersedes original separate Head class approach)
  [x] Write model.py FeedForward class -- two linear layers (N_EMBD -> 4*N_EMBD -> N_EMBD)
      with ReLU activation and dropout
  [x] Write model.py Block class -- layer norm + multi-head attention + residual connection,
      then layer norm + feedforward + residual connection
  [x] Write model.py MiniGPT class -- token embedding, positional embedding, N_LAYER stacked
      Blocks, final layer norm, linear language model head projecting to VOCAB_SIZE logits,
      forward pass, and autoregressive generate method with multinomial sampling
